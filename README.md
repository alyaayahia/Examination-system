# 📚 Examination System – Console‑based C# Application

[![C#](https://img.shields.io/badge/C%23-12.0-blue)](https://docs.microsoft.com/en-us/dotnet/csharp/)
[![.NET](https://img.shields.io/badge/.NET-8.0-purple)](https://dotnet.microsoft.com/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

> A modular, object‑oriented examination system built with C#. Create subjects, design exams, collect answers, and evaluate results – all from the console.

---

## ✨ Features

- Create a **Subject** (ID + name)  
- Dynamically build an **Exam** (Final, Practical, etc.) inside the subject  
- Display questions to the user and capture answers  
- Fully object‑oriented architecture (easy to extend)  
- Clean console output with screen clearing between phases
